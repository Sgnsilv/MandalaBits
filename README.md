# Mandalabits

Mandalabits is a personal productivity web application inspired by bullet journals and physical planning boards. It combines long-term planning, weekly task management, habit tracking, and daily reflections into a single, clean interface.

## ✨Features

- 📅 **Monthly Calendar**
  - Visualize tasks, events, and deadlines
  - Color-coded categories (subjects, projects, etc.)

- ✅ **Weekly To-Do**
  - Simple and flexible task list
  - Manual organization for day-to-day execution

- 🔁 **Habit Tracker**
  - Track daily habits with a visual grid
  - Monthly reset with history preservation
  - Consistency tracking

- 💤 **Sleep Tracking**
  - Log daily sleep hours
  - Visual charts for analysis

- 📝 **Daily Journal**
  - Write notes about your day
  - Reflect and keep a personal record

- 📦 **Archive System**
  - Access past months (e.g., January 2026)
  - View habits and journal entries from previous periods

---

## 🧠 Concept

Mandalabits is based on the idea of organizing life through small, consistent actions — "bits" — that together build a structured and meaningful routine.

---

## 🛠️ Tech Stack (planned)

- **Frontend:** Next.js + React + TypeScript
- **Styling:** Tailwind CSS
- **Backend & Database:** Supabase (PostgreSQL)
- **Charts:** Recharts
